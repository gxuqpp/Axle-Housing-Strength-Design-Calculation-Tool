📐截面计算公式（点击展开） ▶
◆ 圆环截面（式5-57/58） 惯性矩: Iz = Iy = π(D₁⁴ − d₁⁴)/64 式5-57: 弯曲截面系数 Wz = Wy = π(D₁⁴ − d₁⁴)/(32D₁) (mm³) 式5-58: 扭转截面系数 Wp = π(D₁⁴ − d₁⁴)/(16D₁) = 2Wz (mm³) （Wz=Wy，垂直/水平弯曲截面系数相同）
◆ 矩形空心截面（材力公式 + Bredt薄壁闭合） 外宽 B_r × 外高 H_r，内宽 b=B_r−2t_b，内高 h=H_r−2t_h 垂直弯曲（绕水平z轴）： Iz = (B_r×H_r³ − b×h³)/12 Wz = Iz/(H_r/2) 水平弯曲（绕垂直y轴）： Iy = (H_r×B_r³ − h×b³)/12 Wy = Iy/(B_r/2) 扭转（Bredt薄壁闭合截面）： Am = ((B_r+b)/2)×((H_r+h)/2) t_min = min(t_h, t_b) Wp = 2×Am×t_min
◆ 三工况强度校核公式（式5-56/57/58） 工况一（牵引/制动力最大）： Fz2 = m₂'G₂/2， Fx2 = Fz2·φ σ = Mv/Wv + Mh/Wh（Mv=Fz2·b，Mh=Fx2·b） τ = Tт/Wт（Tт=Fx2·rr） σn = √(σ²+4τ²) ≤ [σ] 工况二（侧向力最大，侧滑φ₁=1.0）： Fz₂ₒ = G₂(0.5+hg·φ₁/B₂)（外侧） Fz₂ᵢ = G₂ − Fz₂ₒ（内侧） σ₁ = Fz₂ₒ·(b+φ₁·rr)/Wv（外侧，应力较大） σ₂ = Fz₂ᵢ·|b−φ₁·rr|/Wv（内侧） 工况三（不平路面，动载冲击）： σ = k·G₂·b/(2·Wv) ≤ [σ]
